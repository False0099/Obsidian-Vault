	5. 什么是对象
		1. 一组变量和相关方法的集合，变量表明了状态，方法表明了行为。
2.类
	1.类是描述对象的基本原型，类是 java 的程序基本单元
	2. 程序中的对象是类的一个实例，是一个软件单元，它由一组结构化的数据和在上面的操作组成
	3. 变量：对象的所知道的状态
	4. 方法：对象的功能单元。
	5. 封装：封装把对象的所有组成部分组合在一起，封装定义程序如何引用对象的数据，封装实际上使用方法将数据隐藏起来，控制用户对类的修改和访问数据的程度
