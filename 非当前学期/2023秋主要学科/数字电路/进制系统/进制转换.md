1. 十进制整数转换为二进制整数
我们通过不断地除二模 2，将模到的结果按照顺序反着写过来，就能拿到我们的答案。
```cpp
while(x){
	st.push_back(x%2);
	x/=2;
}
for(auto i:st){
	cout<<i<<endl;
}
```

2. 十进制小数转化为二进制小数
我们每一次将我们的小数部分乘 2，并顺序统计我们的整数部分，就是我们最后的答案。
